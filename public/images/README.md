# Hero Images

## Required Images

- `hero-desktop.webp` - 1920×1080px, < 300KB
- `hero-mobile.webp` - 1080×1350px, < 300KB

## Optimization Tips

Use tools like:

- [Squoosh](https://squoosh.app/)
- ImageMagick: `convert input.jpg -resize 1920x1080 -quality 80 hero-desktop.webp`
- Online converters with quality settings at 75-85%
